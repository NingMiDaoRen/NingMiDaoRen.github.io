### 1.1 快速认识一个HTML文件

我们通过一个基本代码来认识一下HTML文件

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>I'm a title</title>

</head>

<body>
    Hello world
</body>
</html>
```

![](https://images.gitee.com/uploads/images/2020/0630/141553_4ac46f4b_5550632.png)

声明： 

```html
<!DOCTYPE>
```

HTML有多个版本，通过这个标签可以声明出确切的HTML版本

```html
<html> </html>
```

这个标签告诉浏览器这是HTML代码

```html
<head>
  这里是文档的头部 ... ...
  ...
</head>

```

这个标签是文档的头部信息，可以设置字符编码，网页标题等属性。

```html
<body>
  这里是文档的主体 ... ...
  ...
</body>

```

这个是文档的“身体”，里面写HTML代码。

好了，到此为止，你肯定会会产生一个疑惑，那就是$\color{red}{那一堆<> </>和</>是干什么的有什么作用？}$

#### 0x01 标签的类型与结构

看完了效果，我们来看看标签的结构。大致特点如下：

1. 由尖括号包围的关键词，比如 \<html>

2. 通常是成对出现的 (双标签)，比如 \<p><\p>
3. 标签对中的第一个标签是开始标签，第二个标签是结束标签（闭合标签）
4. 也有单独呈现的标签 (单标签)，如：\<img src="" />



按上述特点，HTML标签按写法可分为2种：

可以叫做：

>  闭合标签: \<h1> \</h1>
>
>  单闭合标签: \<meta/>

![](https://images.gitee.com/uploads/images/2020/0630/145549_528c5ad1_5550632.png)

#### 0x03 HTML样式 

所谓样式，就是一个标签长的模样。我们一共有3种方式来设计样式他们是：

1. 外部样式表
2. 内部样式表
3. 内联样式表

外部样式表指的是，我们可以通过在head标签中嵌套书写link标签的方式从外部引入CSS来修饰HTML

```html
<link rel="stylesheet" href="mystyle.css">
```

内部样式表是，我们把样式写在HTML页面当中,通过在head里嵌套style标签，在一个HTML标签里面写样式

```html
<style type = "text/css">
body {background-color: red}
p {margin-left: 20px}
</style>
```

内联样式表，它就是在标签里写样式。即，通过设置style属性的方式。

```html
<h1 style="color: green">我是 h1 标签</h1>
```

<br>外部样式表举例：

HTML代码：

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Hello World</title>
    <link rel="stylesheet" href="mystyle.css">
</head>

<body >
    <h1>我是 h1 标签</h1>
</body>
</html>
```

CSS代码：

```css
h1{
    color: red;
}

```

最终效果：

### 1.6 HTML表单

所谓的表单就是通过写HTML代码通过HTTP协议的两种请求来向服务器提交数据。

向服务器提交数据的方法有两种，一个是GET，另一个是POST。

我们先展示POST提交表单

表单的标签叫做 **form**

```html
<body>
    <form action="" method="post">
        Username: <input type="text" name="usn"> <br>
        Password: <input type="password" name="pwd"> <br>
        <input type="submit" value="提交">
    </form>
</body>
```

![](https://images.gitee.com/uploads/images/2020/0714/150547_bdd9be58_5550632.png)

输入用户名和密码均为123时，浏览器提交的表单内容。

上述代码中出现了新的标签和属性

form标签中的action属性是跳转属性，通过它就可以跳转到别的页面上。method是提交方式，常用的就是get和post。

input标签：它就是一个输入框，可以根据type属性改变显示的方式text是明文，password是密文，submit是提交。通过name属性可以给每个输入框提交的数据起个别名，在上图中usn和pwd就是我所命的名。通过value可以给输入框定义默认值。

我们接着来看get方式。

```html
<body>
    <form action="" method="get">
        Username: <input type="text" name="usn" > <br>
        Password: <input type="password" name="pwd" > <br>
        <input type="submit" value="提交">
    </form>
</body>
```

![](https://images.gitee.com/uploads/images/2020/0714/151509_e45a93ab_5550632.png)

**GET方法会直接拼接URL，而POST不会去拼接URL。**
