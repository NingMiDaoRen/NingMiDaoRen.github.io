# 分发饼干

## 题目描述

假设你是一位很棒的家长，想要给你的孩子们一些小饼干。但是，每个孩子最多只能给一块饼干。对每个孩子 i ，都有一个胃口值 gi ，这是能让孩子们满足胃口的饼干的最小尺寸；并且每块饼干 j ，都有一个尺寸 sj 。如果 sj >= gi ，我们可以将这个饼干 j 分配给孩子 i ，这个孩子会得到满足。你的目标是尽可能满足越多数量的孩子，并输出这个最大数值。

注意：

你可以假设胃口值为正。
一个小朋友最多只能拥有一块饼干。

## 测试用例

示例 1:

输入: [1,2,3], [1,1]

输出: 1

解释: 
你有三个孩子和两块小饼干，3个孩子的胃口值分别是：1,2,3。
虽然你有两块小饼干，由于他们的尺寸都是1，你只能让胃口值是1的孩子满足。
所以你应该输出1。


示例 2:

输入: [1,2], [1,2,3]

输出: 2

解释: 
你有两个孩子和三块小饼干，2个孩子的胃口值分别是1,2。
你拥有的饼干数量和尺寸都足以让所有孩子满足。
所以你应该输出2.

## 题目翻译

去掉题目中的废话，变成编程语言就是

> 已知条件：
>
> 给你两个数组(向量),一个叫g(孩子)，另一个叫s(饼干)。**【下文中的向量统称为数组】**
>
> > 数据含义：
> >
> > g数组里是**饼干的需求量**，s数组里是**饼干的供给量**。【不考虑饼干可拆分的情况】
> >
> > > 目的：
> > >
> > > 统计有多少个满足供需关系的孩子
> >
> > 坑：未知数组是否有序



## 思路1

##### 通过饼干找小孩

1. 将两个数组排序
2. 从第一块饼干开始，和所有的孩子进行比较
3. 若饼干供给量 >= 孩子需求量,认为这块饼干被孩子吃了，总数【cnt】加一

```c++
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

int findContentChildren(vector<int>& g, vector<int>& s);

int main()
{
	vector<int> g = {7,8,9,10};
	vector<int> s = {5,6,7,8};
	cout << findContentChildren(g,s) <<endl;
	return 0;
}

int findContentChildren(vector<int>& g, vector<int>& s) 
{
	int cnt = 0,g_pos = 0;
	vector<int>::iterator g_it = g.begin();
	sort(g.begin(),g.end());
	sort(s.begin(),s.end());
    for(vector<int>::iterator s_it = s.begin();s_it != s.end();s_it++)
    {
    	for(g_it = g.begin()+g_pos;g_it!=g.end();g_it++)
    	{
            /*注意：每当一个人吃完，他就不应该再参加比较【应从下一个人开始】，这个就是g_pos的目的*/
    		if(*s_it >= *g_it)
    		{
    			cnt++;
    			g_pos++;
    			break;
			}
		}
	}
	return cnt;
}

```

时间复杂度：O(n²)

## 思路 2

##### 通过小孩找饼干【贪心算法】

因为分配饼干这件事之间没有关联，所以每次只看需求量最小的孩子是否被满足即可。

```c++
#include<iostream>
#include<vector>
#include<algorithm>
using namespace std;

int findContentChildren2(vector<int>& g, vector<int>& s);
int main()
{
	vector<int> g = {7,8,9,10};
	vector<int> s = {5,6,7,8};
	cout << findContentChildren2(g,s) <<endl;
	return 0;
}

int findContentChildren2(vector<int>& g, vector<int>& s) 
{
	int cnt = 0;
	vector<int>::iterator g_it = g.begin(),s_it = s.begin();
	sort(g.begin(),g.end());
	sort(s.begin(),s.end());
	
	while(g_it != g.end() && s_it != s.end())
	{
		if(*g_it <= *s_it)
		{
			cnt++;
			g_it++;
		}
		
		s_it++;
	}
	return cnt;
}
```

时间复杂度：O(n)
