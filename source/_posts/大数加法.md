# 大数加法

## Part 1 大数加法存在的意义

&emsp;由于编程语言用来存储数字的数据类型有存储上限，我们需要一种方法来进行大数之间 (例如1000位以上的数字) 的运算。

## Part 2 思路

&emsp;灵感来源生活，我们人在计算加法时通常是这样的

![人工加法](https://images.gitee.com/uploads/images/2019/1225/221625_16e547b2_5550632.png)

![人工加法表格模拟](https://images.gitee.com/uploads/images/2019/1225/221642_d6acad4c_5550632.png)

那么我们只要让计算机模拟这个过程就可以了(把数倒过来再相加)

![计算机加法模拟](https://images.gitee.com/uploads/images/2019/1225/221652_a3c04662_5550632.png)

所以整个算法流程是

1. 将数字读入字符数组中

2. 将其反着写进整型数组里

3. 找到这两个字符数组的长度最大值，以确保每一位都会相加，没有漏项

4. 计算进位问题

5. 从后向前扫描，找到第一个非0值

6. 打印

   -------------

## Part 3  代码实现

```c++
#include<iostream>
#include<cstring>


using namespace std;
#define MAX 10000+10
void add(string s1,string s2)
{
	int na[MAX] = {0},nb[MAX] = {0};
	int max_l = 0,i;
	int l_s1 = s1.size(),l_s2 = s2.size();
	
	max_l = l_s1>l_s2?l_s1:l_s2;
	//倒序存储 
	for(i = 0;i<l_s1;i++)
	{
		na[l_s1-1-i] = s1[i]-'0';
	}
	for(i = 0;i<l_s2;i++)
	{
		nb[l_s2-1-i] = s2[i]-'0';
	}
	//加和进位
	for(i= 0;i<max_l;i++)
	{
		na[i] += nb[i];
		if(na[i]>=10)
		{
			na[i+1] += na[i]/10;
			na[i] %= 10;	
		}	
	} 
	//寻找第一个非零值
	for(i = max_l*2;i>=0 && na[i]==0;i--);
	//开始打印
	for(;i>=0;i--)
	{
		cout << na[i];
	}
}
int main()
{
	string s1,s2;
	cin >> s1>>s2;
	add(s1,s2);
	return 0;
}
```
